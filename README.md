# Airtable

Airtable is a flexible and powerful database solution for managing various types of data.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install the Airtable library.

```bash
npm install || npm i --legacy-peer-deps
```

## Available Scripts

In the project directory, you can run:

```bash 
npm run dev
```
### `npm run dev`
Runs the app in development mode.\
Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

The page will reload when you make changes.\
You may also see lint errors in the console.

```bash 
npm run build
```
### `npm run build`
Builds the app for production to the `dist` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.\
Your app is ready to be deployed!

See the [Vite Deployment Guide](https://vitejs.dev/guide/static-deploy.html) for more information.
